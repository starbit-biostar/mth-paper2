# Data Dictionary — MTH Paper 2

## ecdo-sites.json
56 ancient astronomical/megalithic sites extracted from ECDOview (ecdoview.com).

| Field | Type | Description |
|-------|------|-------------|
| name | string | Site name |
| lat | float | Latitude |
| lon | float | Longitude |
| cultural_group | string | Assigned cultural group (9 groups) |

## ecdo-mth-correlation-analysis.md
Full correlation analysis between MTH cultural TW modes and astronomical site density.
Includes: Spearman ρ, bootstrap CIs, leave-one-out sensitivity, confound tests, astronomical cycle comparison table.
